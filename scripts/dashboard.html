예약 발행 자동화 (Cloudflare Deploy Hook)

                    

                    

                        매일 KST 9시·12시·15시·18시·21시에 GitHub Actions가 Cloudflare 빌드를 트리거합니다.

                        draft: false + 미래 pubDatetime으로 저장하면, 해당 시간 이후 빌드 시 자동 노출됩니다.

                        1회 설정 필요: Cloudflare Pages → Settings → Deploy Hooks에서 Hook URL 발급 후 GitHub Secrets에 CF_DEPLOY_HOOK_URL로 등록
                    

                    

                        
Deploy Hook 설정 방법

                        
    
                            
  1. Cloudflare 대시보드 → Workers & Pages → weblog 프로젝트 클릭
    2. 
                            
  2. Settings → Builds & Deployments → Deploy Hooks → Add Deploy Hook
    3. 
                            
  3. Hook 이름: schedule-trigger, 브랜치: main → URL 복사
    4. 
                            
  4. GitHub 저장소 → Settings → Secrets and variables → Actions
    5. 
                            
  5. New secret: 이름 CF_DEPLOY_HOOK_URL, 값에 Hook URL 붙여넣기
    6. 
                            
  6. 로컬 .env 파일에도 CF_DEPLOY_HOOK_URL=<URL> 추가 (아래 버튼 사용 시 필요)
    7.
